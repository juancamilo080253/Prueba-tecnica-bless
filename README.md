# Proyecto Web con Laravel
## Descripción

Este proyecto corresponde a la implementación inicial de una aplicación web desarrollada con Laravel.
El enfoque principal fue la estructuración del proyecto, organización de carpetas y construcción de las vistas (frontend), aplicando buenas prácticas del framework.
Durante el desarrollo se priorizó:

- Organización clara de la arquitectura.
- Separación de responsabilidades.
- Uso adecuado del sistema de vistas de Laravel.
- Preparación del proyecto para futura implementación de lógica backend (CRUD).

## Tecnologías Utilizadas
- PHP/Laravel
- Blade (Motor de plantillas)
- HTML5
- CSS3
- Bootstrap

## Estructura del Proyecto

La estructura sigue el estándar de Laravel, destacando los siguientes directorios:

- resources/views/ → Vistas organizadas por módulos.
- routes/web.php → Definición de rutas web.
- app/Http/Controllers/ → Controladores base preparados.
- Se implementó una organización modular de vistas para facilitar la escalabilidad y mantenimiento del proyecto.

## ⚙️ Instalación y Ejecución

1️⃣ Clonar el repositorio.

2️⃣ Desde la raíz del proyecto, ejecutar:

php artisan serve

3️⃣ Acceder desde el navegador:

http://127.0.0.1:8000

## 📌 Estado del Proyecto

Actualmente el proyecto se encuentra en fase inicial, enfocado en la estructura y diseño del frontend.
La base del sistema está preparada para integrar funcionalidades dinámicas y operaciones CRUD en futuras iteraciones.
